Дополнительные сведения о работе с программой "Автосервис".
На главном экране во вкладке "Клиентская база" представлен список всех клиентов, во вкладке "Клиент" представлена полная информация о выбранном клиенте.
Для изменения информации о клиенте требуется выбрать клиента из списка и нажать кнопку "Изменить".
Для добавления автотранспорта необходимо сделать двойной клик по соответствующей строке с клиентом из общего списка. Появится новое окно со списками грузовых и легковых автомобилей клиента. Для изменения информации об автотранспорте(после добавления) необходимо сделать двойной клик по соответствующей строке из списка авто.
